# lid_driven_cavity

## Numerical methods

+ Fractional step method
+ WENO scheme
+ TVD RK3

## Eigen package REQUIRED!

See [Eigen](https://eigen.tuxfamily.org/) for more information about Eigen.

## make

Change the INC in makefile to the Eigen directory.

## ./run.sh arg1 arg2
 
 arg1: mesh number along each direction
 
 arg2: Re number
